# 5. Project Development Phase

This phase covers the core ML development work, broken into 5 Epics:

- **Epic 1 — Data Collection and Architecture Design**: Download the dataset, define application architecture.
- **Epic 2 — Visualizing and Analysing the Data**: Import dataset, perform Univariate, Bivariate, and Multivariate analysis.
- **Epic 3 — Data Pre-processing**: Handle missing values, balance the dataset, scale data, split into train/test sets.
- **Epic 4 — Model Building**: Build and compare Decision Tree, Random Forest, KNN, and XGBoost models; evaluate and save the best model.
- **Epic 5 — Application Building**: Build HTML pages and a Flask Python script, then run the application.

Each sub-folder contains the notebooks/scripts for that epic.
